## 理论基础

| 理论                                                  | 用途                             |
| :---------------------------------------------------- | :------------------------------- |
| 矩阵计算 [ 线代 基础 ]                                | 几乎所有计算                     |
| 空间几何 [ 欧式/黎曼几何(椭圆几何) 基础标量向量计算 ] | 坐标转换、投影、拓扑、曲线生成等 |
| 数值分析 [ 插值 ]                                     | 曲线曲面生成等                   |
| 计算机图形学                                          | 研究数字合成和操纵视觉内容       |
